# Link

Link allows users to move from page to page, underneeth the hood it renders an `<a>` element.


## Sizes

Link component comes in two sizes, the default one and the small one.

  <Story name="Default Size" args={{children: "Default Size"}}>

  </Story>
  <Story name="Small Size" args={{size: "small", children: "Small Size"}}>

## Disabled

Link component can be disabled by passing `disabled` prop.
