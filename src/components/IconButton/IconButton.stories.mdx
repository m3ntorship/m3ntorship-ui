# IconButton

Icon Button allows Users to perform actions.

## Variants

Users can change variant of the icon button, by passing `variant` prop.

  <Story name="Default" args={{children: <Star />}}>
    {Template.bind({})}
  </Story>
  <Story name="Ghost" args={{variant:"ghost", children: <Star />}}>

  </Story>
 <Story name="Text" args={{variant:"text", children: <Star />}}>

  </Story>
