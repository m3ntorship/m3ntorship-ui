# Avatar

Avatar show the user image or a text in a circle.

## Variants

You can use the avatar as image or as text.

###  Text Avatar

fot the text avatar, You should pass the first character of first name/last name. it's recomended to just pass 1 or 2 characters

You can use it in 3 sizes : sm | md | lg
