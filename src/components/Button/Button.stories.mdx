# Button

Button allows Users to perform actions.

## Variants

Users can change variant of the button, by passing `variant` prop.

<Canvas>
  <Story name="Default">{Template.bind({})}</Story>
  <Story name="Ghost" args={{variant: "ghost", children: "Ghost"}}>{Template.bind({})}</Story>
  <Story name="Text" args={{variant: "text", children: "Text"}}>{Template.bind({})}</Story>
</Canvas>
