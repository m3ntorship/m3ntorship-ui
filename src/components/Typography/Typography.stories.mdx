# Typography

Typography component makes it easier to use default set of typography styles.

## Variants

Users can change the style of the typography component by passing `variant` prop.

## Semantic Element

Users can change the semantic element of the typography component by passing `component` prop.

```jsx
// This will be rendered as `p`
<Typography component="p">
  Paragraph: Almost before we knew it, we had left the ground.
</Typography>

// This will be rendered as `h1`
<Typography component="h1">
  Paragraph: Almost before we knew it, we had left the ground.
</Typography>
```
